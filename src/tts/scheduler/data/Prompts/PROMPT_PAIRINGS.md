# Prompt Pairings Guide

## Optimal Pose + Prop Combinations

### Police Baton Pairings

**Best poses for baton:**
- `hands_on_hips_authority` + `police_baton_resting_on_thigh`
- `one_hand_on_belt_hip_popped` + `police_baton_held_naturally`
- `arms_crossed_weight_shifted` + `police_baton_behind_back`
- `low_angle_commanding` + `police_baton_held_naturally`

**Why:** Baton complements power poses, resting positions feel authentic to authority role.

### Handcuffs Pairings

**Best poses for handcuffs:**
- `finger_hook_belt_loop` + `handcuffs_clipped_to_belt`
- `lean_in_intimacy` + `handcuffs_held_loosely`
- `one_leg_forward_zipper_centered` + `handcuffs_wrapped_around_fingers`
- `seated_elegant_intimate` + `handcuffs_held_loosely`

**Why:** Handcuffs work best in relaxed, confident stances. Keep positioned low (belt level or below).

### Louboutin Heels Pairings

**Best poses for Louboutins:**
- `three_quarter_over_shoulder` + slight knee bend
- `one_leg_forward_zipper_centered` + one foot subtly forward
- `lean_in_intimacy` + elegant leg positioning
- `hands_on_hips_authority` + slight knee bend

**Camera angles:** Eye-level or slightly lower angle to emphasize heel length and red sole visibility.

### Social Media Specific Pairings

**Instagram carousel hero:**
- `hands_on_hips_authority` + `police_baton_resting_on_thigh` + `louboutin_black_stiletto`
- Clean, confident, fashion-forward

**VIP/Exclusive content:**
- `seated_elegant_intimate` + `handcuffs_held_loosely` + `louboutin_pointed_toe_glossy`
- Sophisticated, intimate, premium
- `seated_stool_kissing_baton` + `police_baton_held_naturally` + `louboutin_black_stiletto`
- Flirtatious tease, white stool, VIP content

**Snapchat/personal:**
- `snapchat_selfie_flirty` + `empty_hands` + (any footwear)
- Authentic, personal connection, no props needed

**Edgy/noir basement:**
- `moody_basement_wall` + `police_baton_held_naturally` + `louboutin_elegant_high`
- Mystery, authority, fashion edge

**Car window dynamic:**
- `leaning_through_car_window` + `empty_hands` + `louboutin_pointed_toe_glossy`
- Street style, dynamic framing

## Tone Control Pairings

**Instagram-safe combinations:**
- Any pose + `empty_hands` or `police_baton_resting_on_thigh`
- Tone: confident, stylish, editorial, fashion-forward
- Avoid: handcuffs for brand-safe content

**Edgy editorial combinations:**
- Power poses + `police_baton_held_naturally` or `handcuffs_held_loosely`
- Tone: assertive, powerful, flirtatious but controlled, high-fashion authority
- Props add edge without crossing into fetish territory

## Carousel Sequencing Suggestion

**6-slide carousel pattern:**
1. Hero: `hands_on_hips_authority` + `police_baton_resting_on_thigh`
2. Detail: `hands_belt_detail_crop` + `handcuffs_wrapped_around_fingers`
3. Attitude: `one_hand_on_belt_hip_popped` + `empty_hands`
4. Full body: `three_quarter_over_shoulder` + `police_baton_held_naturally`
5. Lifestyle: `leaning_through_car_window` + `empty_hands`
6. VIP/Tease: `seated_stool_kissing_baton` + `police_baton_held_naturally`

**Why:** Mix of hero, detail, lifestyle, and tease content creates engaging narrative.

## Quick Reference

**Key principle:** One prop maximum per image. Props are styling elements, not performance pieces. Never obstruct face or body silhouette.
